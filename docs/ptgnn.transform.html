Uses a BFS approach to connect only few stereocenters with maximal effect - not really working for molecules with more than 3 stereo centers. Approach was cancelled(difference in diastereomers is too small) so never further developed.

Separates the permutation trees by introducing new nodes and substituting subtrees in trees with new nodes. Initialized nodes with indices. Edge index is not changed as not used in Permutation tree model. Note that operation is IN-PLACE.

Parameters:

Returns:
