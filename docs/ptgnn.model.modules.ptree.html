class ptgnn.model.modules.ptree.advanced_tree_layer.AdvancedPermutationTreeLayer(k: int, hidden_dim: int, batch_norm: bool = False, use_separate_inv: bool = False, apply_p_elu: bool = True)[source]

Bases: Module

Advanced permutation tree invariant layer. Advanced as it is significantly more efficient than its basic counterpart . Uses order/instruction matrices instead of recursive calls to reduce (slow) python control flow.

__init__(k: int, hidden_dim: int, batch_norm: bool = False, use_separate_inv: bool = False, apply_p_elu: bool = True)[source]

Init function of the advanced permutation tree layer

Parameters:

  • k (int) – Number of elements to set into context at once

  • hidden_dim (int) – hidden dimension

  • batch_norm (bool) – Whether to use the batch norm or not

  • use_separate_inv (bool) – Add separate modules for S and Z type nodes - specified through S2 and Z2 in the permutation tree definition.

  • apply_p_elu (bool) – Whether to use ELU between the two linear layers realizing a p type layer
