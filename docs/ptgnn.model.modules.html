Init function of CustomGPSLayer

Parameters:

  • hidden_dim (int) – hidden dimension

  • local_model (str) – local model of the GPS layer, i.e. to wrap with a residual connection

  • local_model_params (dict) – parameters to pass to the init of the local model

  • global_model (str) – global model of the GPS model - not implemented, ignore it. (Original model had option to use a global model, i.e. Transformer and it was not clear whether this will be added or not).

  • dropout (float) – dropout rate

  • norm_type (str) – defines which norm to use. Either batch or layer

  • kwargs – Auxiliary parameters
